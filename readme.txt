Add simple local pick up selection and delivery date time on checkout page

== Description ==
Required: Woocommerce
Tested: Woocommerce 3.x

Only work default woocommerce shipping method (flat_rate, free_ shipping and local_pickup).

Add time slot at admin page under wocommerce-> JW Simple Localpickup and Delivery.
It will show on checkout page, admin order, customer order and email.

== Installation ==
1. Go to your admin area and select Plugins -> Add new from the menu.
2. Search for "JW simple localpickup and delivery".
3. Click install.
4. Click activate.
